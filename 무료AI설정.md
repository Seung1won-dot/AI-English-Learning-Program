# 🆓 무료 AI(Groq) 연결 가이드

OpenAI는 유료지만, **Groq**를 쓰면 **무료**로 진짜 AI 피드백을 받을 수 있습니다.
Groq는 OpenAI와 호환되는 API라서 이 프로그램에 그대로 붙습니다.
(키만 넣으면 자동으로 무료 AI 모드로 전환됩니다.)

---

## 1. Groq 가입 & 키 발급 (2분)

1. https://console.groq.com 접속
2. Google 계정 등으로 **무료 가입** (카드 등록 필요 없음)
3. 왼쪽 메뉴에서 **API Keys** 클릭
4. **Create API Key** 클릭 → 이름 아무거나 입력 → 생성
5. 나온 키(`gsk_...`)를 **복사** 합니다.
   > ⚠️ 키는 이때 한 번만 보입니다. 안전한 곳에 복사해 두세요.

---

## 2. 프로그램에 키 넣기

1. 프로젝트 폴더의 `.env.example` 을 복사해 이름을 `.env` 로 바꿉니다.
   ```
   copy .env.example .env
   ```
2. `.env` 파일을 메모장으로 열고, `GROQ_API_KEY=` 줄에 복사한 키를 붙여넣습니다:
   ```
   GROQ_API_KEY=gsk_여기에_복사한_키
   ```
3. 저장하고 닫습니다. **끝!**

---

## 3. 실행 & 확인

```
python app.py
```

→ 브라우저에서 http://127.0.0.1:5000 접속

화면 위쪽 배지가 이렇게 바뀌면 성공입니다:

```
🤖 Groq (무료 AI) 실시간 피드백 사용 중
```

이제 영어 문장을 쓰면 **진짜 AI(Llama 모델)**가 교정·피드백·점수를 줍니다.

---

## 💡 참고

- **키를 안 넣으면** 그대로 규칙기반 자동 피드백으로 동작합니다 (에러 안 남).
- **우선순위:** `GROQ_API_KEY`(무료) → `OPENAI_API_KEY`(유료) → 목업
  - 즉, Groq 키가 있으면 무료 AI를 우선 사용합니다.
- **모델 바꾸기:** `.env` 의 `GROQ_MODEL` 값으로 조절
  - `llama-3.3-70b-versatile` — 품질 좋음 (기본값)
  - `llama-3.1-8b-instant` — 더 빠름, 가벼움
- **무료 한도:** 개인 학습용으로 충분합니다. 한도를 넘으면 잠깐 기다렸다 다시 됩니다.
- Groq가 잠시 안 되면 자동으로 목업 피드백으로 대체되어 학습이 끊기지 않습니다.

## 다른 무료 옵션 (참고)

| 방법 | 비용 | 특징 |
|------|------|------|
| **Groq** (현재 적용) | 무료 | 빠름, 코드 호환, 가입만 하면 끝 |
| Google Gemini | 무료 한도 | 한국어 품질 좋음, 전용 코드 필요 |
| Ollama (로컬) | 무료 | 인터넷·키 불필요, 내 PC에 설치 필요(8GB+) |

원하면 Gemini나 Ollama 연결도 추가할 수 있습니다.
