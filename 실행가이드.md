# 🚀 English World 실행 가이드

영어 학습 프로그램을 처음부터 실행하는 방법을 순서대로 정리했습니다.
**컴퓨터를 잘 몰라도 따라 할 수 있게** 한 줄씩 적었습니다.

---

## 0. 준비물 (한 번만)

- **Python 3.10 이상** 이 설치되어 있어야 합니다.
- 확인 방법: 명령창에서 아래 입력 → 버전이 보이면 OK
  ```
  python --version
  ```
  → 예: `Python 3.11.9`

> ❓ "python을 찾을 수 없습니다" 라고 나오면 → https://www.python.org 에서
> Python을 설치하세요. 설치 시 **"Add Python to PATH"** 체크박스를 꼭 켜세요.

---

## 1. 명령창(터미널) 열기

이 프로젝트 폴더에서 명령창을 엽니다.

**가장 쉬운 방법 (Windows):**
1. 파일 탐색기에서 `D:\project\english world` 폴더를 엽니다.
2. 주소창을 클릭하고 `cmd` 라고 입력한 뒤 **Enter**.
3. 그 폴더 위치에서 검은 명령창이 열립니다.

또는 직접 이동:
```
cd "D:\project\english world"
```

---

## 2. 필요한 라이브러리 설치 (한 번만)

명령창에 아래를 입력하고 **Enter**:

```
pip install -r requirements.txt
```

→ Flask, OpenAI 등 필요한 것들이 자동으로 설치됩니다.
(인터넷 연결 필요, 1~2분 정도 걸릴 수 있어요.)

---

## 3. (선택) OpenAI API 키 넣기

> ⚠️ **이 단계는 건너뛰어도 됩니다.**
> 키가 없어도 프로그램은 **규칙기반 자동 피드백**으로 그대로 작동합니다.
> 진짜 GPT 피드백을 받고 싶을 때만 하세요.

1. 폴더 안의 `.env.example` 파일을 복사해서 이름을 `.env` 로 바꿉니다.
   - 명령창으로 하려면:
     ```
     copy .env.example .env
     ```
2. `.env` 파일을 메모장으로 열고, 아래 줄에 본인 키를 붙여넣습니다:
   ```
   OPENAI_API_KEY=sk-여기에_본인_키_붙여넣기
   ```
3. 저장하고 닫습니다.

> 🔑 키는 https://platform.openai.com 에서 발급받을 수 있습니다.

---

## 4. 프로그램 실행

명령창에 아래를 입력하고 **Enter**:

```
python app.py
```

이런 메시지가 나오면 성공입니다:
```
 * Running on http://127.0.0.1:5000
```

---

## 5. 브라우저에서 사용하기

1. 크롬·엣지 같은 브라우저를 엽니다.
2. 주소창에 아래를 입력하고 **Enter**:
   ```
   http://127.0.0.1:5000
   ```
3. **연습하기** 화면에서 영어 문장을 한 줄 쓰고 **[피드백 받기]** 클릭!
   - 교정 문장 + 한국어 피드백 + 점수가 나옵니다.
   - (단축키: `Ctrl + Enter` 로도 제출 가능)
4. 위 메뉴의 **학습 기록** 을 누르면 날짜별 기록과 통계를 볼 수 있습니다.

---

## 6. 종료하는 법

- 명령창에서 **`Ctrl + C`** 를 누르면 서버가 꺼집니다.
- 다시 켜려면 4번(`python app.py`)만 다시 하면 됩니다.

---

## ❓ 자주 묻는 문제

| 증상 | 해결 |
|------|------|
| `python`을 찾을 수 없음 | Python 설치 + "Add to PATH" 체크 후 명령창 다시 열기 |
| `pip`을 찾을 수 없음 | `python -m pip install -r requirements.txt` 로 대신 실행 |
| 5000 포트가 이미 사용 중 | `app.py` 맨 아래 `port=5000` 을 `port=5001` 등으로 변경 |
| 화면이 안 열림 | 명령창에 `Running on ...` 메시지가 떴는지 먼저 확인 |
| 한글이 깨져 보임 | 브라우저에서는 정상입니다. (명령창 글자 깨짐은 무시해도 됨) |
